3D Printed LED Lamp

I wanted to expand the list of microcontrollers that I had experience with and I saw that the ESP8266 was more capable than an Arduino, had WiFi connectivity, and came at a cheaper price. I had also recently purchased a 3D printer and wanted to try my hand designing models in CAD and printing them to quickly itterate on an idea. Putting these two things together, I came up with a geometric lamp design and soon enough I had a working prototype in my hands.

  • ESP8266 Microcontroller
  • Neopixel 12 LED Ring
  • Momentary Push Button

CAD Model Design

This was my first time using CAD programs to do any projects and it was a big learning experience for me. I started with TinkerCAD, which was the simplest to get working with, but this project required a bit more precision so I moved to Fusion360 which can be used for free for personal projects. Here the CAD diagram is shown, the pentagonal base is made up of 3 pieces, and the lamp shade is a single piece.

3D Printer

I had heard about 3D printers in my schooling at Arizona State University but I had never really had any hands on time with them so I decided to purchase one for myself. I learned a lot about the machine during this project, previously, I hadn't realized just how precise it could be and the level of calibration that needs to be done for it to work properly.

Software

The ESP8266 is at the heart of the design running C code which controls the LEDs and monitoring the button which allows the user to change colors. Currently, the lamp just does some static colors, but in the future I intend on linking the lamp to the internet so that its color can change in relation to some information on the web (I am thinking weather). The user presses down on the top of the lamp (which depresses the button at the bottom) thus triggering a state change in the software.
