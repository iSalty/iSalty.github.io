Desk Backlight

The Desk Backlight project is powered by a Raspberry Pi and a 1 meter long LED strip. Shortly after moving to Michigan for my new job at Ford I wanted to spruce up my apartment and decided that some backlights for the desk would do the trick. They can do static colors or enter a mode which is able to mimic the colors present on the monitor screens. Below is a diagram of the project components and wiring.

Button Debounce

I included a momentary push button for the user to change modes and turn ON/OFF all of the LEDs. Initially I ran into issues where a single button press would register as mulitple because of button bounce. I wrote extra software logic to reject button presses that occur within 0.2s of the previous to improve performance.

Client & Server Programs

In order to get the data from my desktop computer to my Raspberry Pi I need a program running on each device. The server side program (on my PC) screenshots the monitors twice per second, downsamples the data, and uploads the color data to a website. The client side program (on the R-Pi) retrieves the data from the website and upates the LEDs accordingly using a serial communication interface to the Dotstar LED strip.
